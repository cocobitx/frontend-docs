# Definición

SEO(Search Engine Optimization, o Optimización para Motores de Búsqueda) es el conjunto de técnicas y estrategias que se usan para mejorar la visibilidad y el posicionamiento de un sitio web en los resultados orgánicos de buscadores como Google, Bing o DuckDuckGo.

### Cómo funciona realmente

El funcionamiento del SEO se basa en cómo los motores de búsqueda rastrean, indexan y clasifican el contenido:

1. **Rastreo (Crawling)**:
Los motores de búsqueda usan “bots” o “crawlers” que exploran la web siguiendo enlaces, leyendo páginas y recopilando información sobre su contenido.
 
2. **Indexación**:
La información recopilada se almacena en una gigantesca base de datos llamada índice. Solo el contenido que está correctamente indexado puede aparecer en los resultados de búsqueda.

3. **Clasificación (Ranking)**:
Cuando un usuario hace una búsqueda, el motor revisa su índice y usa un algoritmo para decidir qué páginas mostrar y en qué orden. Este algoritmo considera cientos de factores, entre ellos:

    - **Contenido**: Relevancia y profundidad del contenido respecto a la búsqueda.
    - **Palabras clave**: Qué tan bien coinciden las palabras de la búsqueda con las del sitio.
    - **Experiencia del usuario**: Velocidad de carga, diseño móvil, facilidad de navegación.
    - **Autoridad**: Popularidad de la página y del dominio, medida principalmente por enlaces desde otros sitios.
    - **Intención de búsqueda**: Qué tipo de contenido busca el usuario (informativo, comercial, local, etc.).

4. **Actualización continua**:
Los algoritmos de Google y otros buscadores cambian constantemente. El SEO no es un proceso único, sino un trabajo continuo para mantener o mejorar posiciones.

### Claves del SEO efectivo:

- **SEO On-Page**: Optimización dentro del sitio: estructura del contenido, etiquetas HTML, velocidad, enlaces internos, meta descripciones, etc.
- **SEO Off-Page**: Factores externos al sitio, principalmente enlaces entrantes (backlinks) y menciones de autoridad.
- **SEO Técnico**: Aspectos que facilitan el rastreo y la indexación, como mapas del sitio (sitemap), archivos robots.txt, URLs limpias y seguridad (HTTPS).
- **SEO de Contenido**: Crear contenido de calidad que responda a lo que buscan los usuarios.
