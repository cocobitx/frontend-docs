# Niveles de aplicación
Cada tecnología está orientada a un nivel específico de la aplicación. Su elección depende de los requerimientos funcionales, de rendimiento y de plataforma, ya que cada una ofrece ventajas particulares según el contexto de uso y el tipo de solución a desarrollar.

![logo](/public/aplicaciones.png)

- Public Web Applications: Aplicaciones web que requieran SEO
- Lading Page: Web estaticas, por lo general para sitios corporativos
- P/P Web Applications: Aplicaciones que no requieren SEO, estas pueden ser publicas o privadas, tiene integraciones para transformar la web en aplicaciones de escritorio con Tauri o moviles con Ionic.

::: info
Astro se puede integrar con Vue3, si se requiere paginas estaticas con
rapidez y poca logica
:::
